pandas get_dummies #

pandas 中的 get_dummies 方法主要用于对类别型特征做 One-Hot 编码（独热编码）⁴。One-Hot 编码的基本思想是将离散型特征的每一种取值都看成一种状态，若你的这一特征中有 N 个不相同的取值，那么我们就可以将该特征抽象成 N 种不同的状态，One-Hot 编码保证了每一个取值只会使得一种状态处于“激活态”，也就是说这 N 种状态中只有一个状态位值为 1，其他状态位都是 0³。

get_dummies 方法的参数如下²：

  • data：array-like, Series, or DataFrame，需要进行编码的数据。
  • prefix：str, list of str, or dict of str, default None，给编码后的列名添加前缀。
  • prefix_sep：str, default '_'，如果添加前缀，使用的分隔符/分界符。
  • dummy_na：bool, default False，是否增加一列来表示空缺值，如果 False 则忽略空缺值。
  • columns：list-like, default None，指定需要编码的列名，如果为 None 则对所有具有对象、字符串或类别类型的列进行编码。
  • sparse：bool, default False，是否使用 SparseArray 来支持编码后的列，如果为 True 则使用，如果为 False 则使用普通 NumPy 数组。
  • drop_first：bool, default False，是否去除第一个级别，从 k 个分类级别中得到 k-1 个虚拟变量。
  • dtype：dtype, default np.uint8，新列的数据类型。

例如：

python
import pandas as pd
import numpy as np
s = pd.Series(list('abca'))
print(pd.get_dummies(s))
# output:
#    a  b  c
# 0  1  0  0
# 1  0  1  0
# 2  0  0  1
# 3  1  0  0

s1 = ['a', 'b', np.nan]
print(pd.get_dummies(s1))
# output:
#    a  b
# 0  1  0
# 1  0  1
# 2  0  0

print(pd.get_dummies(s1, dummy_na=True))
# output:
#    a  b  NaN
# 0  1  0    0
# 1  0  1    0
# 2  0  0    1

df = pd.DataFrame({'A': ['a', 'b', 'a'], 'B': ['b', 'a', 'c'], 'C': [1,2,3]})
print(df)
print(pd.get_dummies(df))
# output:
#    C  A_a  A_b  B_a  B_b  B_c
# 0  1    1    0    0    1    0
# 1  2    0    1    1    0    0
# 2  3    1    0    0    0    1

print(pd.get_dummies(df, prefix=['col1', 'col2']))
# output:
#    C  col1_a  col1_b  col2_a  col2_b  col2_c
# 0  1       1       0       0       1       0
# 1  2       0       1       1       0       0
# 2  3       1       0       0       0       1

scipy.stats.norm sklearn.preprocessing.StandardScaler scipy.stats #

python
from scipy.stats import norm
from sklearn.preprocessing import StandardScaler
from scipy import stats

  1. scipy.stats.norm 是 scipy 库中的一个模块，用于生成正态分布或高斯分布的概率密度函数和累积分布函数。可以用它来计算正态分布的一些统计量，例如均值、标准差、概率密度函数值、累积分布函数值等。

  2. sklearn.preprocessing.StandardScaler 是 sklearn 库中的一个模块，用于将数据缩放到均值为 0，方差为 1 的标准正态分布中。它对数据进行标准化处理，可以使不同特征的数据在同一尺度上进行比较，提高模型的精度和稳定性。

  3. scipy.stats 是 scipy 库中的一个模块，包含了很多统计学上的方法和分布。它提供了一些常用的统计函数，例如 t 检验、方差分析、卡方检验等，可以用来进行数据分析和统计建模。

pandas drop #

pandas drop方法有以下参数²：

  • labels：要删除的行或列的标签，可以是单个标签或列表。
  • axis：要删除的轴，0表示行，1表示列，默认为0。
  • index：要删除的行的标签，等同于axis=0。
  • columns：要删除的列的标签，等同于axis=1。
  • level：对于多级索引，要删除的索引层级。
  • inplace：是否在原数据上进行操作，如果为True，则返回None，否则返回一个副本，默认为False。
  • errors：如果为'ignore'，则忽略不存在的标签，如果为'raise'，则抛出错误，默认为'raise'。

使用方法如下：

python
df.drop(labels=None, axis=0, index=None, columns=None, level=None, inplace=False, errors='raise')

例如，如果要删除列A和B，可以这样写：

python
df.drop(columns=['A', 'B'], inplace=True)

或者

python
df.drop(labels=['A', 'B'], axis=1, inplace=True)
